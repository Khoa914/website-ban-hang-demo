# Website bán hàng (E-commerce Demo)

Dự án website bán hàng được xây dựng để thực hành phát triển web full-stack cơ bản.

## Công nghệ sử dụng
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Chức năng chính
- Đăng ký / đăng nhập
- Xem danh mục sản phẩm
- Thêm sản phẩm vào giỏ hàng
- Thanh toán
- Quản lý sản phẩm (admin)

## Mục đích
Dự án thực hành trong quá trình học ngành Công nghệ Thông tin.
